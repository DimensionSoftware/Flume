Flume Music App
===============

[http://flumemusic.com](http://www.flumemusic.com)
--------------------------------------------------

## Quick Start

    $ git clone git@github.com:DimensionSoftware/Flume.git
    $ cd flume && npm i && npm start
    $ open ios/flume.xcodeproj/

  Press Command+R in Xcode to Build &amp; Run!
